Задача 1: «Обмен данными» (Тип int и Арифметика)
1.	Создайте/Объявите две переменные a и b типа int и сразу присвойте им значения (например, a = 5 и b = 10).
2.	Выведите начальные значения.
3.	Используйте только арифметические операции (+ и -), чтобы поменять их значения местами (чтобы a стало 10, а b стало 5).
4.	Выведите конечные значения.
Пример логики (не код!):
1.	a = a + b (a = 15)
2.	b = a - b (b = 15 - 10 = 5)
3.	a = a - b (a = 15 - 5 = 10)
Ожидаемый вывод:
Начало: a=5, b=10
Конец: a=10, b=5

Задача 2: «Пересчёт метров» (Типы int, double и /)
Пользователь вводит общее число сантиметров. Ваша задача — перевести это в полные метры и оставшиеся сантиметры.
1.	Создайте/Объявите переменную total_cm типа int.
2.	Считайте значение от пользователя (например, 234).
3.	Используйте оператор деления (/) для нахождения полных метров.
4.	Используйте оператор остатка (%) для нахождения оставшихся сантиметров.
5.	Выведите результат.
Пример работы:
Введите сантиметры: 234
>> Это 2 полных метра и 34 сантиметра.

Задача 3: «Расчет среднего балла» (Деление с точкой)
Пользователь вводит три целые оценки. Нужно посчитать точный средний балл (с десятичными знаками).
1.	Объявите три переменные типа int для оценок (g1, g2, g3).
2.	Считайте эти три оценки через cin.
3.	Объявите переменную average типа double.
4.	Вычислите сумму оценок, а затем разделите ее на 3. Обязательно используйте приведение типов (например, (double)) или число с точкой (3.0), чтобы избежать целочисленного деления.
5.	Выведите результат.
Пример работы:
Введите 3 оценки (например, 5 4 4):
>> 5 4 4
>> Средний балл: 4.33333
